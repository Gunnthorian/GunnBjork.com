2-Axis Camera Motion System

This project involved the development of a dual-axis camera slider designed to automate complex motion for time-lapse photography. Built around an Arduino microcontroller, the system manages two independent stepper motors through dedicated drivers: one to translate the jig along a linear rail and another to rotate the camera, keeping the subject centered. This setup demonstrates a successful implementation of multi-axis synchronization, translating manual cinematography techniques into an automated, programmable hardware solution.

CNC Router Build

This project highlights the design and assembly of a CNC router built by scratch and developed specifically for the rapid prototyping of a custom skateboard project. The machine features an Arduino-based control system paired with a motor shield to run GRBL, an open-source high-performance G-code interpreter. Even with a Dremel utilized as the primary spindle, the build successfully demonstrates the ability to translate digital CAD models into physical components with precision and efficiency.
